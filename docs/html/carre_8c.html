Description détaillée

Fichier de fonctions pour accesseurs et de modifications pour un Carre et une Piece.

Auteur
WIDMER Alexis
Version
1.0
Date
01/03/2019

Ce fichier permet d'avoir accès aux attributs d'un Carre et d'une Piece, de les créer, de les supprimer ou de les afficher.
